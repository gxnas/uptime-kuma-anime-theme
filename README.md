# 🎨 Uptime Kuma Anime Theme

一个 **Uptime Kuma 的二次元主题切换包**，内置三种风格：  
- 🌸 **默认二次元主题**（粉蓝渐变，梦幻风格）  
- 🌌 **暗色二次元主题**（赛博夜景，霓虹风格）  
- ☀️ **浅色二次元主题**（粉蓝清新，白天风格）  

右下角有一个 🎨 按钮，可以一键切换主题，自动记忆上次选择。  

---

## 🚀 使用方法

### 1. 引入 JS 文件
在 Uptime Kuma → **设置 → 外观 → 自定义 HTML** 添加：

```html
<script src="https://cdn.jsdelivr.net/gh/gxnas/uptime-kuma-anime-theme@latest/anime-theme.js"></script>
